# 💼 Employee Payroll Management System

A full-stack web application to manage employee data, payroll, and attendance with role-based access (Admin, HR, Employee).

## 🚀 Features

* Employee Management
* Payroll Generation
* Attendance Tracking
* Role-based Authentication

## 🛠️ Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB

## ⚙️ Run Locally

```bash
npm install
npm start
```
